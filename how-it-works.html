üîê The Encryption Process

1

Select Your Image

Choose any image file (JPG, PNG, GIF, WebP, BMP) from your device. The image stays on your computer - nothing is uploaded.

2

Create a Strong Password

Enter a secure password. We recommend at least 12 characters with mixed case, numbers, and symbols. This password will be required to decrypt your image later.

3

Automatic Encryption

Your browser uses AES-256-GCM encryption to protect your image. A random salt and initialization vector ensure each encryption is unique.

4

Download Encrypted File

Receive a .pwimg file that contains your encrypted image. This file can only be opened with the correct password.

üîì The Decryption Process

1

Select Encrypted File

Choose your .pwimg file. The file format is verified to ensure it's a valid encrypted image.

2

Enter Password

Provide the same password used during encryption. The password is converted to an encryption key using the same secure process.

3

Automatic Decryption

Your browser decrypts the image using the same AES-256-GCM algorithm. The process verifies data integrity.

4

View or Download

Your original image is displayed or downloaded. You can also copy it to your clipboard for immediate use.
